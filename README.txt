Quickstart
----------

./loxodo.py
	runs Loxodo in GUI mode (if available)

./loxodo.py -h
	displays help about Loxodo's cmdline mode

./setup.py py2app
	creates a stand-alone .app (Mac OS X) 

./setup.py py2exe
	creates a stand-alone .exe (Windows)

./loxodo.py -i
	runs Loxodo in CLI Interactive mode
